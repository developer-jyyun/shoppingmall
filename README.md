# React + TypeScript + Vite를 사용한 쇼핑몰 웹사이트
- Firebase를 이용한 OAuth 로그인, 실시간 데이터베이스 사용  
- Cloudinary 서비스를 이용한 제품의 이미지 업로드 기능
- admin 계정 사용자, 제품 정보 추가, 수정, 삭제 CRUD 기능
- 소비자들의 장바구니 추가 / 삭제 기능
